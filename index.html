I also grew significantly in my understanding of data structures and algorithms. Revolv 360 Weight Loss Tracker

Revolv 360 is a mobile weight tracking app that lets users log daily weights, view their history, see progress badges, and enroll in motivational alerts when they reach their goals.

  • HTML
  • CSS
  • JavaScript
  • Android Studio
Visit Site →

For my software engineering and design enhancement, I updated my Android Weight Tracker application, which was originally created in CS 360. The first version met the functional requirements for the course, but the interface and architecture were simple and reflected my early development skills. For CS 499, I upgraded the entire UI using Jetpack Compose and redesigned the app to look and function like a modern mobile product. This involved replacing XML layouts with composable structures, refining spacing and alignment, redesigning widget cards, and adding a seven day progress badge that calculates weekly weight changes. I also reorganized the project into clearer, reusable components and improved state management to make the interface more consistent and responsive. Through this enhancement, I demonstrated advanced skills in modern Android development, including building interfaces with Jetpack Compose, designing user friendly layouts, structuring application state, and applying UI and UX principles to improve usability. I strengthened my ability to refactor code for readability and long term maintainability, worked through layout and preview issues in Android Studio, and ensured that the new UI integrated smoothly with existing login, navigation, and data storage features. The redesign required both technical reasoning and creativity, helping me develop a more professional approach to front end engineering. This enhancement aligns with the course outcome related to using well founded and innovative techniques, skills, and tools to implement computing solutions. By modernizing the application with Jetpack Compose and improving its architectural clarity, I demonstrated my ability to design software that is scalable, maintainable, and aligned with current industry standards. The updated interface now reflects my growth as a developer and represents the level of quality I aim to bring into my future career as a front end software engineer.
